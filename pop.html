Популярность ОС MS Windows

NetMarketShare: минимальные колебания на рынке ОС

Свежие данные от NetMarketShare показывают, что в прошлом месяце доля самой популярной ОС для десктопной платформы, Windows 10, практически не изменилась. Несмотря на релиз May 2019 Update, аналитическая компания зафиксировала лишь минимальное увеличение адаптации Windows 10 – с 45,73% до 45,79%.



Microsoft стала поставлять Windows 10 May 2019 Update (версия 1903) в конце мая, но только в июне обновление стало доступно для ручной установки всем пользователям. Для запуска установки любой желающий может зайти в Центр обновления Windows, нажать кнопку «Проверить наличие обновлений», а затем среди доступных обновлений найти Обновление функций до Windows 10, версия 1903 и выбрать опцию «Загрузить и установить сейчас».


Доля Windows 7, поддержка которой завершится в январе 2020 года, в июне сократилась с 35,44% до 35,38%. Microsoft завершит обслуживание системы 2009 года 14 января будущего года, поэтому рекомендует пользователям этой системы обновиться до Windows 10, чтобы продолжить получать обновления безопасности. Переход с Windows 7 на Windows 10 осуществляется очень медленными темпами. До сих пор 3 из 10 компьютеров по всему миру по-прежнему работают под управлением Windows 7.


Третьей по популярности ОС для десктопной платформы остается Windows 8.1, чья доля в июне увеличилась с 3,97% до 4,51%.


Windows XP уходит в прошлое

Windows XP, которая перестала получать обновления в апреле 2014 года, наконец, стала стремительно терять аудиторию. На данный момент система 2001 года работает на 1,81% ПК по всему миру. В основном, Windows XP намеренно используется в организациях по причинам совместимости и больших издержек на обновление до современных версий Windows.


Linux, которая считается одной из главных альтернатив Windows, по состоянию на конец июня работает на 1,55% десктопных компьютеров.



AdDuplex: лишь единицы перешли на May 2019 Update

Пользователи Windows 10 неохотно устанавливают May 2019 Update, даже несмотря на отсутствие в обновлении серьезных ошибок. По всей видимости, ситуация с October 2018 Update, наложила отпечаток на поведение пользователей. Еще одна возможная причина – изменение политики развертывания обновлений.


В результате адаптация версии 1903 происходит очень медленными темпами, которые примерно соответствуют темпам развертывания предыдущего обновления – October 2018 Update.


Новые статистические данные компании AdDuplex показывают, что доля October 2018 Update (версия 1809) в июне сократилась с 31,3% до 30%, а доля May 2019 Update увеличилась с 1,4% до 6,3%.



Таким образом, прирост доли версии 1903 за первый летний месяц составил около 5 процентов. В выборке AdDuplex анализировалось около 100 000 компьютеров. Данные темпы обновления соответствуют версии 1809 после ее повторного релиза. Это не лучший признак, потому что October 2018 Update едва смогло достичь 30%-ной доли перед релизом последующего обновления функций.


Примерно половина пользователей May 2019 Update (около 3 процентов) обновились с April 2018 Update — это сигнал того, что многие пользователи не желают продолжать оставаться на версии годичной давности и хотят получить новые функции и улучшения.



Подавляющее большинство ПК Windows 10 (58%) по-прежнему работают на April 2018 Update. А это значит, что разработчикам приходиться поддерживать три основные версии Windows 10, что подрывает идею Microsoft о «Windows как службе».


Steam: системы Windows – почти единогласный выбор геймеров

За июнь 2019 года доля операционных систем Windows среди компьютеров с установленным клиентом игровым клиентом Steam увеличилась на целых 0,59% до более чем убедительных 96,49%. В то же время доля систем macOS сократилась на 0,51%, а доля систем Linux на 0,08%. Во втором случае отток пользователей может быть связан с новостями о прекращении поддержки 32-битных пакетов в Ubuntu. Однако, впоследствии Canonical предложила компромиссное решение, которое устроило Valve. Таким образом, Steam на Ubuntu 20.04 LTS будет поддерживаться до 2025 года.



Геймеры Steam все активнее переходят на 64-битную версию Windows 10 – в июне доля системы выросла на 3,08% до 70,92%. Другими словами, 7 из 10 игроков Steam используют Windows 10. Все остальные системы Windows теряли аудиторию среди игроков сервиса Valve. Так доля Windows 7 сократилась на 2,09% до 21,32% а Windows 8.1 – на 0,18% до 2,74%.
