Загальна інформація
Windows — узагальнююча назва операційних систем для комп'ютерів, розроблених корпорацією Microsoft. Перші версії не були повноцінними операційними системами, а лише оболонками до ОС MS-DOS. На 2014 рік, за даними сайтів NetApplications та GoStats, Microsoft Windows встановлена більш як на 90% персональних комп'ютерів світу.
